# Nutrifit Backend

This is the backend for the Nutrifit project.

## Directory Structure

- **manage.py**: The command-line utility for administrative tasks.
- **nutrifit/**: The main application directory for configurations and settings.
- **nutrifit/settings.py**: The settings for the Django project.
- **nutrifit/urls.py**: URL configurations for the project.
- **nutrifit/wsgi.py**: WSGI configuration for deployment.
- **app/**: Placeholder for Django apps to be created.

## Getting Started

To get started, make sure you have Django installed and then run the server:

```bash
python manage.py runserver
```
